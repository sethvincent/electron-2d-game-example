# electron 2d game example

"hey, can i put a canvas game in an electron app." 

"yeah, i guess so."

## Try it out:

### 1. clone this repo

```
git clone git@github.com:ngoldman/electron-hello-world.git
```

### 2. `cd` into the repo directory and run `npm install`

```
cd electron-hello-world
npm install
```

### 3. run `npm start`

```
npm start
```
